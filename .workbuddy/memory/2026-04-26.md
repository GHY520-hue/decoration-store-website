## 22:34 - 网站内容不显示问题彻底修复

### 根本原因
**API 地址硬编码为 `http://localhost:8080/api`**，Cloudflare Pages 上没有后端服务，导致：
- 首页"推荐案例"和"热销材料"数据为空（依赖 API 获取）

- 页面结构存在但内容区域空白，看起来像"不显示"

   - `src/data/products.ts` - 8个装修材料产品（地板、涂料、灯具、瓷砖、橱柜、卫浴、门窗等）

   - Home.vue / Products.vue / Projects.vue / ProductDetail.vue / ProjectDetail.vue
